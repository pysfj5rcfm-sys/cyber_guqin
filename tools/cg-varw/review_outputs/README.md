# CG-VARW Review Outputs

This directory is for local review-only R0/R1 drafts and CSV exports.

Generated files under `r0/drafts/`, `r0/exports/`, `r1/drafts/`, and `r1/exports/` are intentionally ignored by git.

All generated artifacts here are:

```text
review_only=true
production_grade=false
not_sample_assets=true
not_render_executed=true
not_ml_training_data=true
```

They are not sample ingest data, not recording segments, not sample assets, not render output, and not ML training data.
