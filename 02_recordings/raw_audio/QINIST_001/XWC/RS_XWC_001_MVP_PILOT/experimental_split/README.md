# Experimental Split Sandbox

Session: `RS_XWC_001_MVP_PILOT`

This directory contains only non-production experimental split artifacts for
the MVP pilot. It is not a formal sample library and does not register segment
candidates into `03_samples`.

ffmpeg was used for working WAV conversion with ffmpeg_pcm_s16le_wav. The source files remain M4A/AAC experimental raw material and do not become production sources.
